ข้อมูลหลักสูตร

ชื่อหลักสูตร: วิชาวิศวกรรมสารสนเทศและเครือข่าย

รหัสหลักสูตร: 06105

ระดับการศึกษา: ปริญญาตรี

คณะ/มหาวิทยาลัย: คณะเทคโนโลยีและการจัดการอุตสาหกรรม / มหาวิทยาลัยพระจอมเกล้าพระนครเหนือ

คำอธิบายหลักสูตร: วิศวกรรมสารสนเทศและเครือข่าย เป็นหลักสูตรระดับปริญญาตรี 4 ปี ที่เน้นการเรียนรู้ด้านเทคโนโลยีสารสนเทศและเครือข่าย โดยมีการเสริมทักษะภาษาอังกฤษและเน้นการเรียนรู้ที่ทันสมัย เช่น Cloud Computing, SDN, Automation, Machine Learning, Blockchain, Big Data, BI, Image Processing, IoT, UX, และ Decision Thinking
